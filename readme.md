#### 如何使用？
```bash
# 依次执行以下指令
git clone https://github.com/bennypeng/Laravel5.5_Common.git Project1
cd Project1
composer install 
cp .env.example .env # 并修改里面的配置项
php artisan key:generate
php artisan cache:clear
php artisan migrate
chmod -R 777 storage/
php artisan laravels start
```

#### 并没有加入什么新奇玩意，仅用于工作方便使用
